# Webpack4-Vue-Template

这是一个基于Webpack4、Element-UI、Vue的项目模板项目，目的是以后可以快速的创建基本的项目基本框架，以及开发过程中的最佳实践

- 打包构建工具：Webpack4
- 前端MVVM框架：Vue2.x
- UI框架：Element-UI
- 路由：Vue-Router
- 状态管理：Vuex
- ajax库：axios
- 日期格式化库：dayjs
- 工具库：lodash
